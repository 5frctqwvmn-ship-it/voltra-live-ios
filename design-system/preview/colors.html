Color Palette

A 4-surface dark canvas, one accent (teal), and four semantic phase colors. Reverse-mapped from VoltraTheme.swift.

Surface ramp

Canvas--vl-bg#0a0e0c
Page background. Almost-black with a green hint.
Surface--vl-bg-elev#11181a
Tiles, sheets, primary cards.
Nested--vl-bg-elev-2#1a2426
Chips inside tiles, hover, pressed.
Hairline--vl-border#1f2c2e
1px borders around every elevated surface.

Text on canvas

Primary text --vl-text · #e8f4f1
Big readouts, headlines, row text.
Dim text --vl-text-dim · #8aa39e
Tile labels, secondary copy, units, meta.
Faint text --vl-text-faint · #4a5f5b
Placeholders, disabled, never-data.

Accent + phase

Accent --vl-accent · #00d4aa
Brand teal. PULL phase, primary CTAs, connection state, the icon. Used sparingly.
Accent dim --vl-accent-dim · #007a62
Pressed / muted accent state.
Pull (concentric) --vl-pull · #00d4aa
PHASE tile + force chart segment when going up.
Return (eccentric) --vl-return · #ffb84d
PHASE tile + force chart segment when going down.
Transition --vl-transition · #6c8de0
Between-rep transition. Force chart only.
Idle --vl-idle · #4a5f5b
Phase = idle. No active set.

Status

Warn --vl-warn · #ff7a4d
Recoverable: BT off, no telemetry for 2 s, low battery.
Danger --vl-danger · #ff4d6d
Hard errors. The BOTTOM marker on the force chart. Reserved.
Fresh data --vl-fresh · rgb(51,217,102)
Pulse dot when HealthKit data is < 5 s old. Distinct from accent on purpose.
Stale data --vl-fresh-stale · rgb(89,89,89)
Pulse dot when data has gone stale.

Phase washes (12% tints — used on phase tile only)

Pull wash--vl-pull-washrgba(0,212,170,.12)
Return wash--vl-return-washrgba(255,184,77,.12)
Transition wash--vl-transition-washrgba(108,141,224,.08)
