VOLTRA Live · Design System

v0.1 · dark-only · iOS 17+ · iPad-landscape primary

A black instrument panel with one teal phosphor color and big mono numerals. Built for a sweaty user mounting a phone or iPad on a cable rack and reading it from 8 ft. This page indexes the tokens, components, and principles.

Principles

01
Operator, not coach
Report state. Don't motivate. No emoji, no exclamation marks. The user is the coach.
02
Numbers do the talking
Big mono tabular numerals on flat tiles. Labels are short, all-caps, +2 px tracked. The page is a readout, not a dashboard.
03
One color carries meaning
Teal accent for connect / pull / primary. Phase colors are semantic. Warning ≠ danger. No color is decorative.
04
Flat surfaces, hairline edges
No gradients, no shadows on tiles. A 1 px #1f2c2e hairline is the only edge treatment.
05
Built for a sweaty thumb
Tap targets ≥ 44 px. Primary CTAs 50 px. Whole tiles tappable at 56+ so a wet thumb mid-set always lands.
06
Four tiles, no more
Live screen is REPS · PHASE · FORCE · REST. A 5th tile means removing one first. Density is a feature.
